# Finance-CS50
A project of CS50 task

# How to run:

(IMPORTANT)  
Run following commands in terminal  
```
clone https://github.com/tanze1233/Finance-CS50.git  
cd Finance-CS50  
virtualenv venv  
. venv/bin/activate  
pip install cs50,flask,flask_session  
```
  
  
After it completes, you can either run  
```
flask run  
```
or  
```
python application.py  
```
to start the server.
